# Kuantist V.1.0 - Gelişmiş Web Arayüzü

Kuantist, React ve yapay zeka teknolojileri kullanılarak geliştirilmiş modern bir web asistanıdır.

## 🚀 Özellikler

- **🧠 Çoklu Yapay Zeka Desteği:** 
  - **Groq Entegrasyonu:** Llama-3 modelini kullanarak saniyeler içinde zeki yanıtlar verir.
  - **İnternet Araması:** Tavily API ile güncel internet bilgilerini tarar.
  
- **🎨 Görsel Oluşturma:** 
  - "Resim çiz" diyerek veya `/img` komutuyla anında yüksek kaliteli görseller oluşturun.

- **💬 Çoklu Sohbet Yönetimi:** 
  - Farklı konular için yeni sohbetler başlatın.
  - Sohbet geçmişinizi tarayıcı hafızasında saklar.

- **🖌️ Kişiselleştirilebilir Temalar:** 
  - Indigo, Emerald, Violet, Amber gibi profesyonel renk paletleri arasında geçiş yapın.

- **📂 Dosya ve Medya Analizi:** 
  - Görsel yükleyerek içeriği hakkında soru sorun.

## 🛠️ Kurulum

1. `npm install` komutu ile bağımlılıkları yükleyin.
2. `npm run dev` ile projeyi başlatın.
3. Ayarlar menüsünden API anahtarlarınızı (Groq & Tavily) girin.

## 🔑 API Anahtarları

Uygulamanın tam fonksiyonel çalışması için aşağıdaki anahtarlara ihtiyacınız vardır:
- **Groq API Key:** Metin üretimi ve sohbet için.
- **Tavily API Key:** İnternet aramaları için.
*(Görsel oluşturma özelliği ücretsizdir ve anahtar gerektirmez)*
